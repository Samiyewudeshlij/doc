Introduction

The aim of this project-based web development training is to help students become full stack developers. Web Design, Frontend Development (HTML5, CSS3, Javascript), Backend Development (Node js and Express js), and Database System (MySQL) are all covered throughout this training. Throughout this training, we'll look at a variety of resources, including VS Code, Adobe XD, and wamp.

During this course, we will be creating a Mini-Social Media Web Application that will be able to register users, authenticate users, fetch posts, edit posts, and delete posts.

Upon completion of this training you will be able to understand and work with the tools and technologies mentioned in above.

Who can Participate?

The training is open to those interested in learning Web Development. To begin the journey, there are no prerequisites or previous experience in technology needed.

What are you going to get?

  • You will be able to develop Static and Dynamic Web Apps
  • You will be able to understand Database systems
  • You will have your own Social Media Web App.

What is Expected from you?

  1. Stay Motivated
  2. Try working on your own examples.
  3. Prepare for class by reading ahead.
  4. Ask Questions

About the Project

Our project for this training is a Mini-Social Media Web Application. The mini-Social Media Web App will be supporting multiple users at a time and store data persistently. Features:

  1. Enable users to register using [ Name, Email, Password]
  2. Enable users to Login to the system. Using their Email and Password.
  3. Enable Users to edit their Account
  4. Enable users to create Posts
  5. Enable Users to view Posts
  6. Enable users to edit Posts
  7. Enable Users to delete Posts
  8. Enable Users to Comment on Posts

How are we going to Build the project

The goal is to learn web development while developing this mini-social media web application, so we'll use a development model that enables us to start from the beginning and work our way up to more complex web topics like backend development.

First Phase-Web Designing

The first step in designing every web application is to design how the application will be organized and what will be on each page. In the development and maintenance of websites, web design involves a wide range of skills and disciplines. We will learn about the importance of web design, the elements of web design, and the principles of web design during this phase.

Tools: Adobe XD

Second Phase-Frontend Skeleton

We'll use HTML 5 to build the skeletal structure and layout the elements of the web page in this phase. HTML5 is a markup language that is used to structure and display content on the internet. It is the fifth and final big HTML edition to be recommended by the World Wide Web Consortium.

Tools: HTML 5, VS Code

Third Phase- Frontend Beautify

In this phase, we are going to discuss styling web pages using CSS 3 and Bootstrap 5. CSS is the language we use to style a Web page.

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Tools: CSS 3, Bootstrap 5, VS Code

Fourth Phase- Frontend Behavior

To create a dynamic and interactive experience for the user in our web apps we use a programming language named Javascript. In this phase we will discuss fundamental programming concepts such as variables, conditionals, functions, and Loop.

Tools Javascript ,VS Code

Fifth Phase- Database Design

A database is typically designed so that it is easy to store and access information. A good database is crucial to any web application. This is because the database stores all the pertinent details about our web app such as posts, comments, user data, and so on…

Tools MySQL ,WAMP or XAMPP

Sixth Phase: Backend Development

In this phase we are going to build Learn about Node js and Express js, which we are going to use to build an Application Program Interface (API). The API will help us to communicate between our frontend and backend.

Tools Node js ,Express js, VS Code
